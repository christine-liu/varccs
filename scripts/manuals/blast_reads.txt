BLAST Quality Controlled reads to Reference Exons

Description:

	BLAST quality controlled reads to reference exon sequences, and generate
	tables with exon alignments and exon summaries.

Usage:

	blast_reads
		-i input.fasta
		-e exons.fasta
		-o output_prefix
		[options]

Mandatory arguments:

	-i input.fasta | --input input.fasta
		input.fasta is the FASTA file obtained from the fastq_qc_to_fasta step.

	-e exons.fasta | --exons exons.fasta
		exons.fasta is the FASTA file with reference exon sequences.

The following [options] are available:

	-p output_prefix | --prefix output_prefix
		Output filenames will star with this "output_prefix". Can include full path.
		If not given, output prefix is automatically generated from the input file, e.g.
		if the input file is /path/to/filename.fastq, output_prefix will be
		/path/to/filename, and all output files generated by will be of a form:
		/path/to/filename_*.*

	-ws integer | --word-size integer
		Word size in nucleotides for BLAST alignment.
		Default: 25

	-go integer | --gap-open integer
		Gap opening penatly for BLAST alignment.
		Default: 0

	-ge integer | --gap-extend integer
		Gap extension penalty for BLAST alignment.
		Default: 2

	--fix-lowq-homopolymer-runs 0|1
		0 = do NOT change homopolymer runs to the reference sequence
		1 = do change homopolymer runs to the reference sequence
		Default: 1
